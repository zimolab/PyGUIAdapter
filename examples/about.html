Equation Solver

This application can be used to solve equations in the form:

ax^2 + bx + c = 0 (a,b,c ∈ R and a ≠ 0)

Notes:
  • all inputs are real numbers
  • non-real solutions will be discarded

This is a simple example of PyGUIAdapter，which is a opensource Python project for creating GUI applications from a python function without writing any GUI code. Please visit our GitHub page for more information： PyGUIAdapter.
